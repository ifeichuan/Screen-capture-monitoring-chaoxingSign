<h1 align="center">一个用于监控windows屏幕的小玩具，使用Python+express编写 </h1>

## 环境 💻

`监控.py`文件为Client端,会自动截图屏幕上传到express服务器. 时间间隔为2.5秒 可自行修改

`app.js`为express服务端，运行在3000端口上
